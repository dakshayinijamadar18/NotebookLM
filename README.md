# 📓 NotebookLM AI Assistant

An AI-powered knowledge workspace inspired by Google NotebookLM.  
This app helps users upload, organize, and interact with documents using AI.

---

## 🚀 Features

- 📄 Upload and view documents
- 🤖 AI-powered chat with your notes
- 🧠 Smart summarization
- 🔍 Context-aware answers
- 🗂️ Organized workspace (like notebooks)
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

- Next.js
- TypeScript
- React
- Tailwind CSS
- AI APIs (OpenAI / others)
